# Stock Mate Frontend

React-based frontend for Stock Mate - Personal Investment Intelligence Platform

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- Vite
- TanStack Query (React Query)
- Zustand for state management

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_URL=http://localhost:8007
VITE_WS_URL=ws://localhost:8007
```

## Design System

Following Toss-style design principles:
- Clean, minimal interface
- No gradients or AI-like aesthetics
- Primary color: #4056F4 (Full Spectrum Blue)
- Secondary color: #E3B23C (Golden Rod)
- White: #FFFFFF (backgrounds, cards)
- Rest: Grayscale only
